Idris Bomba Family Tree - Shared Cloudinary Photo Version

This version uses fixed Cloudinary public IDs so photos can appear on every phone/computer.

Configured values:
Cloud name: dx0vkjhwf
Upload preset: family-tree-upload
Folder/public ID prefix: idris-bomba-family

IMPORTANT Cloudinary preset settings:
1. Upload preset must be Unsigned.
2. Set Overwrite = true.
3. Set Unique filename = false.
4. Set Use filename = false.

Why these matter:
Every person has a fixed image address, for example:
https://res.cloudinary.com/dx0vkjhwf/image/upload/.../idris-bomba-family/idris_ibrahim

When someone uploads a photo for that person, it replaces that fixed image. That is what lets other devices see the same photo.
